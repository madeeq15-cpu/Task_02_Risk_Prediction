# Credit Risk Prediction

## Project Overview

This Task is aims to predict loan approval status using machine learning techniques. The Loan Prediction Dataset was analyzed, cleaned, and visualized to identify factors that influence loan approval decisions.

## Objective

To build a classification model that predicts whether a loan application is likely to be approved based on applicant information such as income, education, credit history, and loan amount.

## Dataset

* Dataset: Loan Prediction Dataset
* Records: 614
* Features: 13
* Target Variable: Loan_Status

## Project Workflow

1. Data Loading and Understanding
2. Missing Value Handling
3. Data Cleaning and Preparation
4. Exploratory Data Analysis (EDA)
5. Feature Encoding
6. Model Building

   * Logistic Regression
   * Decision Tree Classifier
7. Model Evaluation

   * Accuracy Score
   * Confusion Matrix
   * Classification Report

## Key Insights

* Credit History is one of the most important factors affecting loan approval.
* Applicant Income and Loan Amount influence lending decisions.
* Loan approvals are significantly higher for applicants with a positive credit history.
* Machine learning can assist banks in making faster and more consistent loan approval decisions.
